# FoodJet Non-MVP Backlog

Features and capabilities deferred beyond the MVP release, listed by domain.

## Payments

- Online payments integration (credit/debit cards)
- Apple Pay support
- Google Pay support
- Payment refund flow
- Split payment (partial cash + card)

## Delivery & Geolocation

- Delivery maps with interactive zone drawing
- Polygon-based delivery zones
- Geocoding (address to coordinates)
- Geofencing (automatic zone detection from customer address)
- Delivery fee calculation based on distance/zone

## Courier & Logistics

- Live courier tracking (real-time location on map)
- Route optimization for multi-order deliveries
- Automatic courier assignment (auto-dispatch based on proximity, load, availability)
- Courier earnings dashboard
- Courier shift scheduling

## Communication

- In-app customer chat (customer <-> store operator)
- In-app courier chat (courier <-> store operator)
- Push notifications outside Telegram

## Loyalty & Promotions

- Loyalty program (points accumulation)
- Cashback system
- Bonus balance
- Promo engine: discount codes, percentage/fixed discounts, free delivery, buy-X-get-Y
- Referral program

## Integrations

- ERP integration (1C, SAP, etc.)
- POS integration (iiko, Poster, R-Keeper, etc.)
- Accounting system sync
- Inventory management sync

## Analytics & Reporting

- Advanced BI dashboards (sales, AOV, retention, cohorts)
- Real-time operational dashboard
- Custom report builder
- Export to Excel/CSV

## Customer Experience

- Reviews and ratings (for stores, products, couriers)
- Favorites / wishlists
- Order repeat (one-tap reorder)
- Scheduled / pre-orders (order for a specific time)
- Multi-language support

## Order Management

- Split shipments (partial fulfillment from different locations)
- Order batching for couriers
- Priority orders
- Return / exchange flow
